HTML & CSS Quick Reference Guide

HTML Tags

<p> - Paragraph

This tag defines a paragraph of text.

Example:

<p>This is a sample paragraph.</p>

<h1> - Heading 1

This tag defines the largest kind of heading. It can come in 6 forms.

Example:

<h1>Section Title</h1>

<ul> - Unordered List

This tag defines an unordered (bulleted) list. Items are defined by <li>.

Example:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol> - Ordered List

This tag defines an ordered (numbered) list. Items are also defined by <li>.

Example:

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<div> - Division

This tag defines a division or section in an HTML document. It's often used as a container to store HTML elements for better organization.

Example:

<div class="container">
    <p>Content inside the container.</p>
</div>

<img> - Image

This tag embeds an image in an HTML document.

Example:

<img src="image.jpg" alt="Description of the image">

<li> - List Item

This tag defines a list item within an ordered (<ol>) or unordered (<ul>) list.

Example:

<li>List item content</li>

<input> - Input Field

This tag defines an input field for user input.

Example:

<input type="text" name="username" placeholder="Username">

<br> - Line Break

This tag inserts a single line break to move to the next line.

Example:

Line 1<br>Line 2

<hr> - Horizontal Rule

This tag defines a thematic break in an HTML page, and is seen as a divider line.

Example:

<hr>

CSS Properties

color

Sets the color of text.

Example:

p { color: blue; }

background-color

Sets the background color of an element.

Example:

div { background-color: lightgray; }

font-size

Sets the size of the font.

Example:

h2 { font-size: 24px; }

margin

Sets the outer spacing of an element.

Example:

div { margin: 10px; }

padding

Sets the inner spacing of an element.

Example:

div { padding: 20px; }

border

Customizes the border of an element.

Example:

div { border: 1px solid black; }

text-align

Sets the horizontal alignment of text.

Example:

p { text-align: center; }

font-family

Sets the font family. Options include 'Times New Roman', 'Consolas', 'Arial', and more.

Example:

body { font-family: sans-serif; }
